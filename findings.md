# Findings — Trenh Tech Landing Page

## Design Analysis

### Color Palette
| Name        | Hex       | Usage                        |
|-------------|-----------|------------------------------|
| Soft Cloud  | #DFE9ED   | Buttons, light accents       |
| Nordic Sky  | #A0B4BC   | Text, nav links, logo text   |
| Steel Cloud | #65828D   | Secondary text, borders      |
| Deep Tech   | #233A4A   | Dark sections, gradients     |
| Dark Mode   | #1F2429   | Primary background           |

### Typography
- **Helvetica** — Logotype, body text, paragraphs
- **Almarena** — Section titles/headlines (display font)

### Layout Sections (top to bottom)
1. **Header:** Logo left, nav center (Tjänster, Nyhetsbrev, Projekt, Kontakt), "Boka möte" button right
2. **Hero:** Dark vertical texture background, large "From strategy to innovation" headline
3. **About:** Semi-transparent overlay, description text, two CTA buttons (Våra tjänster, Boka möte)
4. **Motto:** Gradient background, "Code. Cloud. Growth." large centered text
5. **Newsletter:** "Nyhetsbrev" header, 3 image cards in grid
6. **CTA Banner:** "Where technology meets business." large text on gradient
7. **Contact:** Dark bg, "Redo att ta nästa steg?" heading, form (Namn, Mail, Meddelande), image right
8. **Footer:** Logo left, links center, LinkedIn icon right

### Constraints
- No backend/APIs
- Placeholder images used for photos (user will replace later)
- Single-page application with anchor navigation
